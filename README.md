# tic-tac-toe-neon
# 🎮 Neon Tic Tac Toe (AI)

A modern, lightweight **Tic Tac Toe game** built with pure HTML, CSS, and JavaScript — featuring a neon UI, AI opponent, and interactive feedback.

---

## 🚀 Demo

Open the `index.html` file in any browser to play.

---

## ✨ Features

* 🤖 AI opponent (auto move)
* 🌈 Neon-style modern UI
* 🔊 Sound effects on moves & win
* 📳 Mobile vibration feedback
* ⚡ Fast and responsive gameplay
* 🔄 Restart anytime

---

## 🎮 How to Play

* You play as **X**
* AI plays as **O**
* Click on any empty box to make a move
* First player to align **3 in a row** wins:

  * Horizontal
  * Vertical
  * Diagonal
* If all boxes fill → 🤝 Draw
